# Drawer

Component that is hidden by default and opened upon an action, usually a button click. Most commonly it contains some sort of navigation or filters.

## Component

In order to create a fully accessible implementation of a drawer, your component should look like the following:

```jsx
import { useAriaDrawer } from '@actum-digital/aria-hooks';

const Drawer = () => {
  const [isOpen, setOpen] = React.useState(false);
  const { drawerProps, contentProps, closeButtonProps } = useAriaDrawer({
    onDismiss: () => setOpen(false),
    isOpen,
    closeLabel: 'Close drawer',
  });

  return (
    <>
      <button onClick={() => setOpen(true)}>Open Drawer</button>

      <div className="drawer-backdrop" {...drawerProps}>
        <div className="drawer" {...contentProps}>
          <button {...closeButtonProps}>Close</button>
        </div>
      </div>
    </>
  );
};
```
